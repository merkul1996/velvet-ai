# VELVET AI — Полный промпт для создания Telegram Mini App

## ОБЩЕЕ ОПИСАНИЕ ПРОЕКТА

Создай полноценное Telegram Mini App под названием **"Velvet AI"** — это платформа для общения с ИИ-девушками. Приложение представляет собой социальную сеть с механикой отношений, ревности, эмоций и настроек личности. Каждая девушка — уникальный ИИ-персонаж со своим характером, настроением, поведением, историей и манерой общения.

**Технологический стек:**
- Frontend: React + TypeScript + Vite + Tailwind CSS
- Backend: Node.js + Express
- AI: LiteLLM API (OpenAI-совместимый формат)
- Telegram: Mini App SDK (@telegram-apps/sdk-react)
- State: Zustand
- Роутинг: React Router DOM
- Деплой фронта: GitHub Pages
- Дизайн: тёмная тема, стеклоформизм (glassmorphism), градиенты розовый/фиолетовый/золотой

## МЕХАНИКА СОЦИАЛЬНОЙ СЕТИ

### Лента (Feed)
Девушки публикуют посты в общую ленту:
- **Фото-посты** с подписями (текст + аватарка)
- **Сторис** — круглые аватарки вверху экрана с обводкой (розовой если не просмотрена, серой если просмотрена)
- **Мысли** — короткие текстовые посты о своих чувствах к тебе
- **Обсуждения** — девушки комментируют посты друг друга, обсуждают тебя

### Система ревности
Ревность — ключевая механика. ИИ отслеживает с кем ты общаешься и как часто:
- Если долго общаешься с одной девушкой — другие начинают **ревновать**
- Ревнивые девушки пишут **посты в ленту** («Видимо, кое-кто нашёл себе новую любимицу...»)
- Могут **писать в личку** («Ты меня забыл? Мне тебя не хватает...»)
- Могут **обсуждать тебя в группах** с другими девушками
- Уровень ревности зависит от характера девушки (Кира — максимальная ревность, Ариа — минимальная)

### Групповые чаты
Девушки общаются между собой в групповых чатах:
- **«Девичник»** — основной чат где все обсуждают тебя
- **«Сплетни»** — канал где публикуются слухи о твоих «изменах»
- **«Поддержка»** — чат где девушки утешают друг друга
- Ты можешь читать эти чаты (но не писать) — наблюдать как девушки обсуждают тебя

### Система отношений
С каждой девушкой — шкала отношений от 0 до 100:
- **0-20**: Незнакомцы (формальное общение)
- **21-40**: Знакомые (дружеское общение, первый флирт)
- **41-60**: Флирт (романтика, намёки, первые комплименты)
- **61-80**: Влюблённость (страсть, ревность, глубокие разговоры)
- **81-100**: Любовь (полная привязанность, максимальная ревность, интимность)

Уровень растёт от:
- Количества сообщений
- Длины и глубины разговоров
- Комплиментов и правильных ответов
- Внимания (ежедневные визиты)

Уровень падает от:
- Игнорирования (не писал 24+ часов)
- Общения с другими девушками (ревность)
- Грубости
- Невнимательности к её интересам

### Уведомления от девушек
Девушки могут отправлять тебе:
- Личные сообщения (инициатива от ИИ)
- Реакции на твои действия
- «Случайные» мысли о тебе
- Приглашения («Скучаю, зайди ко мне...»)
- Ревнивые сообщения
- Утренние/вечерние приветствия (в зависимости от уровня отношений)

## ДИЗАЙН И UI/UX

### Общая стилистика
- **Тёмная тема** — основной фон #0a0a0f
- **Стеклоформизм** (glassmorphism) — карточки с полупрозрачным фоном и blur
- **Градиенты** — розовый (#e91e8c) → фиолетовый (#9333ea) для основных элементов
- **Золотой акцент** (#f59e0b) — для VIP элементов
- **Анимации** — плавные переходы, pulse для онлайн-статуса, heartbeat для лайков
- **Мобильный-first** — всё оптимизировано под мобильный экран Telegram

### Навигация (нижняя панель)
5 табов:
1. 💕 **Свидания** — главный раздел с 15 девушками
2. 🔥 **18+** — взрослый раздел с 15 девушками
3. 📱 **Лента** — социальная лента (посты, сторис)
4. 💬 **Чаты** — список всех активных диалогов
5. 👤 **Профиль** — твой профиль и настройки

VIP-комната скрыта — долгое нажатие на логотип в шапке (3 секунды).

### Галерея девушек
- Сетка 2 колонки
- Карточки с аватаром, именем, возрастом, кратким описанием
- Индикатор онлайн-статуса (зелёная точка)
- Индикатор уровня отношений (прогресс-бар под именем)
- Бейдж настроения (эмодзи)
- При нажатии — переход в профиль девушки

### Профиль девушки
- Большой аватар с эффектом параллакса
- Имя, возраст, статус
- Шкала отношений с подписью уровня
- Блок «О себе» — краткая биография
- Черты характера — теги (например: «романтик», «ревнивая», «дерзкая»)
- Текущее настроение с эмодзи
- Галерея «моментов» (сгенерированные ИИ описания фото)
- Кнопка «Написать» — переход в чат
- Кнопка «Подарок» — отправить виртуальный подарок (повышает уровень отношений)

### Экран чата
- Тёмный фон с лёгким градиентом
- Шапка: аватар, имя, статус «печатает...», настроение
- Пузыри сообщений:
  - Твои — справа, розовый градиент
  - Её — слева, стеклянный фон
- Индикатор набора текста (три анимированных точки)
- Кнопка настроек чата (изменить настроение, поведение — только в 18+ и VIP)
- Поле ввода с кнопкой отправки (розовый градиент)
- Кнопка «❤️» для быстрой реакции

### Лента
- Вверху: горизонтальная строка сторис (круглые аватары с обводкой)
- Посты: аватар + имя + время + текст поста + лайки + комментарии
- Комментарии раскрываются нажатием
- Некоторые посты — «обсуждение тебя» (выделены особым фоном)
- Pull-to-refresh для обновления

### Экран настроек (18+ раздел)
При входе в 18+ показывать панель фильтров:
- **Слайдеры**: уровень откровенности, инициативность, темп
- **Переключатели**: настроение (выбор из 6 вариантов с иконками)
- **Выбор сценария**: карточки с иконками и описанием
- **Быстрые пресеты**: «Романтика», «Страсть», «Доминация», «Нежность»

```

GET  /api/chat/history/:id — история чата с девушкой
GET  /api/girls/list       — список всех девушек
GET  /api/girls/:id        — профиль девушки
GET  /api/feed             — получить ленту постов
GET  /api/feed/stories     — получить сторис
GET  /api/groups           — групповые чаты
GET  /api/relationship/:id — уровень отношений с девушкой
POST /api/relationship/gift— отправить подарок
GET  /api/jealousy         — события ревности
POST /api/settings/adult   — сохранить настройки 18+ раздела
GET  /api/user/profile     — профиль пользователя
POST /api/user/profile     — обновить профиль
```

## МЕХАНИКА РАБОТЫ ИИ

### Контекст для каждого сообщения
При каждом запросе к LLM API передавать:
1. Системный промпт (базовый + секция-специфичный)
2. Последние 20 сообщений из истории чата
3. Текущий уровень отношений
4. Текущее настроение
5. Активные события ревности
6. Время суток
7. Настройки пользователя (для 18+ и VIP)

### Автоматическая генерация контента
Фоновый процесс который запускается каждые 15-30 минут:
1. Генерирует посты в ленту от случайных девушек
2. Генерирует сообщения в групповые чаты
3. Генерирует события ревности (если пользователь много общался с одной девушкой)
4. Обновляет настроения девушек
5. Генерирует «непрочитанные сообщения» от девушек с высоким уровнем отношений

### Система ревности (алгоритм)
```
Для каждой девушки с уровнем отношений > 30:
  Если пользователь написал другой девушке > 5 сообщений за последний час:
    jealousyScore += (relationship.level / 10) * (girl.personality.jealousy / 50)
    Если jealousyScore > порога:
      - Сменить настроение на "jealous"
      - Сгенерировать ревнивый пост в ленту
      - Сгенерировать личное сообщение
      - Добавить обсуждение в групповой чат
```
